Tech You Need for this Unit

Required:

  • A Computer with:
  • 1.6 GHz Processor
  • 4 GB RAM
  • 120 GB Hard Drive
  • Web Cam
  • Microphone
  • Required Downloads, Installations, and Account setups

Recommended Specs:

  • 2 GHz Processor
  • 8 GB RAM
  • 256 GB Hard Drive (preferably solid state)

Recommended Operating Systems:

  • macOS
  • Windows 10 Pro
  • Ubuntu
Note: Chromebooks are not supported, as coursework involves installing and running software locally.

Attendance and Participation

BloomTech measures engagement through:

  • Live instruction attendance
  • Module & sprint project attempts
  • Learning Management System activity

Important: Learners inactive for more than 7 days will be flagged and contacted for enrollment verification.

BloomTech may withdraw inactive learners to protect them from unintentional tuition vesting.

If you need to take a break due to extenuating circumstances, please request a leave of absence.

How To Pass A Technical Sprint

  • Sprint Challenge: 100% Required
    • Time-bound project using sprint skills
    • Unlimited attempts after first submission
  • Sprint Assessment: 80% Required
    • Multiple-choice knowledge check
    • Unlimited attempts after first submission

How To Pass an ACS Sprint

  • Attempt the General Code Assessment (GCA)
  • Complete the career readiness assignment
