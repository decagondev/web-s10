Technical Requirements

Required Hardware

  • A Computer with:
  • 1.6 GHz Processor
  • 4 GB RAM
  • 120 GB Hard Drive
  • Web Cam
  • Microphone
  • Required Downloads, Installations, and Account setups

Recommended Specifications

Computer Specs

  • 2 GHz Processor
  • 8 GB RAM
  • 256 GB Hard Drive (preferably solid state)

Operating Systems

  • macOS
  • Windows 10 Pro
  • Ubuntu
  • Other versions of Windows work but students should anticipate occasional environment issues.
Chromebooks are not supported as coursework involves installing and running software locally.
